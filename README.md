# Quarto Live Curl Demo

A demo of using the {curl} R package with [Quarto Live](https://r-wasm.github.io/quarto-live/).

Uses [coi-serviceworker](https://github.com/gzuidhof/coi-serviceworker) to enable Cross-Origin Isolation on GitHub Pages, allowing network requests via a WebSocket proxy.

**[View the live demo](https://georgestagg.github.io/quarto-live-curl/)**
